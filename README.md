<h1>Conducting a Network Scan in Nessus</h1>

<h2>Description</h2>
In this project I will create a basic network scan using Nessus.

<h2>Languages and Utilities Used</h2>

- <b>Nessus</b> 


<h3>Environments Used </h3>

- <b>Linux</b>

<h3>Program walk-through:</h3>

<p align="center">

First I'm going to log into Nessus and select Basic Network Scan.

<img src="https://i.imgur.com/SLFIZ8h.png" height="80%" width="80%" alt="Nessus Network Scan"/>

Next I'm going to enter the target ip address.

<img src="https://i.imgur.com/2P81sgX.png" height="80%" width="80%" alt="Nessus Network Scan"/>

Set the scan type to port scan (all ports) then click launch.

<img src="https://i.imgur.com/fmCiwLf.png" height="80%" width="80%" alt="Nessus Network Scan"/>

The scan is currently running.
